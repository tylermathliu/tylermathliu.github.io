欧式空间中的运动（保距变换）一定是仿射变换

证明：设\(E\)是欧式空间，\(f:E\to E\)是保距变换，设\(g(x)=f(x)-f(0)\), 只需证\(g\)是线性. 由于\(f\)是保距，于是有事实：

  1. \(|g(x)-g(y)|=|x-y|"\)
  2. 特别的， \(|g(x)|=|x|​\)

欧式空间的平行四边形公式： \[ |x+y|^2+|x-y|^2=2(|x|^2+|y|^2) \] 我们需要证明两点：

  1. \(g(x+y)=g(x)+g(y)\)
  2. \(g(\lambda x)=\lambda g(x)\)

第一点，考虑式子 \[ I=2|g(x+y)-g(x)-g(y)|^2+|g(x+y)+g(x)-g(y)|^2+|g(x+y)-g(x)+g(y)|^2 \] 一方面，把后两项用平行四边形公式加起来，再利用事实1,2，得到 \[ I=2|g(x+y)-g(x)-g(y)|^2+4(|x|^2+|y|^2) \] 另一方面， 把一二项、一三项用平行四边形公式加起来，利用事实1,2，得到 \[ I=4(|x|^2+|y|^2) \] 于是我们看出 \[ 2|g(x+y)-g(x)-g(y)|^2=0 \] 这证明了第一点； 作为第一点的推论，我们得到，\(g\)保持内积；作为推论，我们容易得到第二点. \(\Box\)

Mazur-Ulam定理

赋范线性空间中的保距双射是仿射变换

设\(E​\)，\(F​\)是实赋范线性空间，\(f:E\to F​\)是一个保距. 于是\(f​\)是一个仿射变换，当且仅当 \[ f(tx+(1-t)y)=tf(x)+(1-t)f(y),\forall t\in [0,1], x,y\in E​ \] 当且仅当 \[ f(\frac{x+y}{2})=\frac{f(x)+f(y)}{2}, \forall x,y\in E​ \] 一般情形下，\(f\)未必是个仿射变换. 但是在下面两种情形下，\(f\)必然是个仿射变换

  1. 若\(F\)是严格凸空间，那么\(f\)是仿射变换.
    这是因为\(f(\frac{x+y}{2})\)是距离，\(f(x)\), \(f(y)\)均为\(|\frac{x-y}{2}|\)的点（因为f是保距）； 而在F中分别以\(f(x)\),\(f(y)\)为球心，\(|\frac{x-y}{2}|\)为半径的球相交于唯一一个点\(\frac{f(x)+f(y)}{2}\). 从而\[f(\frac{x+y}{2})=\frac{f(x)+f(y)}{2}, \forall x,y\in E\] 由于欧式空间自然是严格凸的（平行四边形公式），所以欧式空间是它的一个特例.
  2. 若\(f\)是个满射（从而是双射），那么\(f\)是仿射变换（Mazur-Ulam定理）

在着手证明Mazur-Ulam定理之前，说一下证明大概想要是干什么的应该是有益的.
>设\(g\)是一个赋范线性空间E的保距自双射，如果\(g\)保持某两个点不动，那么g也一定保持这两个点的中点不动 (*)

事实上， \[ f(\frac{x+y}{2})=\frac{f(x)+f(y)}{2}, \forall x,y\in E​ \] 是在说，描述仿射变换的一个方式是，仿射变换把线段的中点映成线段的中点.

于是证明的思路是：在\(E\)中以线段\(\overline{xy}​\)的中点\(z​\)（即\(z=\frac{x+y}{2}​\)）为中心进行反射，记这个反射为\(\psi​\)，那么\(x​\)和\(y​\)恰好把自己映成对方.
同样地，在\(F\)中以\(\overline{f(x)f(y)}​\)的中点为中心反射，记为\(\psi'​\). 设\(h=\psi f^{-1} \psi' f​\), 那么\(h\)是一个保持\(x​\)，\(y​\)不动的保距自双射，从而由那句话，\(hz=z​\), \(\psi' f z=fz​\).
由于\(\psi'​\)是反射，故\(\psi'​\)有唯一不动点\(\frac{f(x)+f(y)}{2}​\), 从而\(f(z)=\frac{f(x)+f(y)}{2}​\), 这即说明了f是仿射变换.

我们下面证明Mazur-Ulam定理：

设\(E\)为线性赋范空间，定义\(E\)中关于点\(z\in E\)反射为\(\psi x:=2z-x\),由定义立得，\(\psi\)是个对合，且对任意\(x\in E\)成立等式 \[ |\psi x-z|=|x-z|, |\psi x-x|=2|x-z| \] 设\(a\),\(b\)是\(E\)中两点，设\(z=\frac{a+b}{2}\), 设\(W\)是所有保持\(a\)，\(b\)不动的\(E\)的保距自双射构成的群，设 \[ \lambda:=\sup \{|gz-z|:g\in W\} \] 设\(g\in W\), 由于 \[ |gz-a|=|gz-ga|=|z-a| \] 故 \[ |gz-z|\leq |gz-a|+|a-z|=2|a-z| \] 故\(\lambda < \infty\).

设\(\psi\)是\(E\)中以\(z\)为中心的反射，若\(g\in W\)，那么 \[ g*:=\psi g^{-1}\psi g\in W \] 从而反复利用关于反射的两个等式得到 \[ 2|gz-z|=|\psi gz- gz|=|g^{-1}\psi gz-z|=|\psi g^{-1}\psi gz-z|=|g*z-z|\leq \lambda \] 从而\(2\lambda \leq \lambda\)，即\(\lambda=0\). 我们证明了(*)的那个结论.

设\(F\)是线性赋范空间，设\(f:E\to F\)是保距，设\(z':=\frac{f(a)+f(b)}{2}\), 设\(\psi'\)F中以\(z'\) 为中心的反射，那么\(h=\psi f^{-1} \psi' f\in W\), 故\(hz=z\), 或者说，\(\psi' fz=fz\) 但\(z'\)是\(\psi'\)唯一的不动点，故\(fz=z'\). \(\Box\)

Lie group and Lie algebra [reproduced]

Posted on 2018-03-05 | In Mathematics |

李群和李代数，是微分几何和代数交叉的精彩领域，似乎在量子力学、计算机视觉等领域有着比较丰富的应用。本文的目的是简单地介绍李群和李代数的最基本概念和应用，我们不追求数学的精确性和严格性，首先我们将主要围绕在典型群的周围来领略旖旎的风光以及一些应用。
首先是一些基本概念：代数方面你只需要知道什么是群就行了，至于几何，你至少要了解直到指数映射。

概念的罗列

这一节可能比较枯燥，不过你也可以先跳过，一遍看第二节的例子，一边在回顾这一节。
一个群\(G\)，如果它同时是一个光滑流形，则称其为李群（Lie group）。
事实上，只要群\(G\)的乘法和逆运算都是光滑映射，则\(G\)成为一个李群。
记1为群\(G\)的单位元，将1处的切空间\(\mathrm{T}_{1}G\)称为李代数（Lie Algebra），记作\(\mathfrak{g}\)（哥特字体）或者\(\mathrm{Lie}(G)\)。
我们知道在黎曼几何中，指数映射是从切空间到流形上的映射，\(\mathrm{exp}_{p}(v)\)指从\(p\)点出发沿切向量为\(v\)的测地线行走\(|v|\)的长度到达的点，

接下来我们将把指数映射推广到李群之上。在李群中，指数映射被限制在单位元1上，即指数映射\(\mathrm{exp}(x)\)为\(\mathfrak{g}\rightarrow G\)的映射。
显然\(\mathrm{exp}(0)=1\)
对于\(x,y\in\mathfrak{g}\)，\(x,y\)足够小的时候，\(\exp(x)\exp(y)\)应该很接近\(1\in G\)。因此我们可以定义： \[\exp(x)\exp(y)=\exp(\mu(x,y))\]
其中 \(\mu:\mathfrak{g}\times\mathfrak{g}\rightarrow\mathfrak{g}\) 为定义在\((0,0)\)附近的光滑映射。
\(\mu\)可以展开泰勒级数： \[\mu(x,y)=x+y+\lambda(x,y)+\cdots\] \(\cdots\)表示高阶项。
其中\(\lambda(x,y)\)是反对称（扭对称）的双线性映射。

再定义\([x,y]=2\lambda(x,y)\)，这个称为李括号。
下面我们给出一个性质：
\[\exp(x)\exp(y)\exp(-x)\exp(-y)=\exp([x,y]+\cdots)\]
这个性质是针对一般的李群而言的，接下来我们将看到，在实际应用所接触到的实用李群中，这个性质起着非常关键的作用。

典型群的例子

现在，把我们的目光将主要集中在一般线性群(\(\mathrm{GL}_{n}(\mathbb{R})\))的子群上。
一般线性群是所有\(n\times n\)的非奇异矩阵以乘法为群运算构成的。显然矩阵乘法和逆都是光滑的，
因此\(\mathrm{GL{n}}\)是一个李群，它的李代数\(\mathrm{Lie}(\mathrm{GL}{n})\)定义为单位矩阵的切空间。它的一些子群称为典型群。
典型群包括正交群（\(\mathrm{O(n)}\)），酉群（\(\mathrm{U(n)}\)），辛群(\(\mathrm{Sp(n)}\))。典型群都是子群。
一般线性群的李代数为所有的n阶矩阵。指数映射在典型群下有着一些很好的特性，即
\[\exp(x)=\sum_{k=0}^{\infty}\frac{x^{k}}{k!}\] 为矩阵指数。
同时若\(xy=yx\) 则 \(\exp(x)\exp(y)=\exp(x+y)\)
注意到这个性质的条件在一般的李代数中是没有定义的，因为李代数中没有定义乘法，而在典型群中，由于李代数是矩阵集合，因此该式具有了意义。

现在我们将典型群中指数映射的重要性质不加证明地写下：

\[\exp(x)=\sum_{k=0}^{\infty}\frac{x^{k}}{k!}\]

\[xy=yx\Rightarrow\exp(x)\exp(y)=\exp(x+y)\]

\[\exp(x^{T})=(\exp(x))^{T}\]

我们先来看正交群\(\mathrm{O(n)}\)。显然\(\mathrm{O(n)}={X|XX^{\mathrm{T}}=I,X\in\mathbb{R}^{n\times n}}\)，令\(x\in\mathrm{Lie(O(n))}\),则
\[\exp(x)=X\]
\[\exp(x^{T})=X^{T}\]
因此
\[\exp(x+x^{T})=XX^{T}=I\]
所以
\[x+x^{T}=0\]
所以正交群的李代数是所有反对称矩阵构成的空间。
考察二维正交群的子群特殊正交群（\(\mathrm{SO(2)}\)），也称为二维旋转群，它的李代数只有一个基，
\[G_{1}=\begin{bmatrix}0\ -1\\1 \ \ \ \ \ 0\end{bmatrix}\]
对于\(\mathrm{alg}(\theta)=\theta G_{1}\in Lie(SO(2))\)，（\(\mathrm{alg}(\theta)\)表示以\(\theta\)为系数对基进行线性组合得到的向量），指数映射为，
\[\exp(\mathrm{alg}(\theta))=\exp(\begin{bmatrix}0 \ -\theta\\\theta \ \ \ \ \ 0\end{bmatrix})=\begin{bmatrix}\cos\theta \ -\sin\theta\\sin\theta \ \ \ \ \ \cos\theta\end{bmatrix}\]

下面我们先来看一个在计算机视觉中可能的应用：
定义对数映射\(\log(X)=x\)为指数映射的逆，利用指数映射和对数映射，我们可以对变换进行插值。假如\(X,Y\in G\)为两个变换，我们想要对它们进行插值\(f(X,Y,t):G\times G\times[0,1]\rightarrow G\)。我们可以沿着两个变换间的测地线进行插值：
\[\begin{aligned}f(X,Y,t) \ ; =\exp(t\cdot\log(Y\cdot X^{-1}))\cdot X\end{aligned}\]
注意到
\[\begin{aligned}f(X,Y,0) \; =X\\f(X,Y,1) \; =Y\end{aligned}\]
接下来，我们来看\(\mathrm{SE(2)}\)，即平面刚性变换群，显然一个刚性变换由一个旋转和一个平移组成。因此
\[\begin{bmatrix}\mathbb{R} \ \ \ \mathrm{t}\\\ 0 \ \ \ 1\end{bmatrix}\in\mathrm{SE(2)\subset\mathbb{R}^{3\times3}}\] 其中\(\mathbb{R}\in\mathrm{SO(2)},t\in\mathbb{R}^{2}\)。
因此它的李代数由三个generators生成：
\[G{1}=\begin{bmatrix}0 -1 \; 0\\1 \ \ \ 0 \ \ \ 0\\0 \ \ \ 0\ \ \ 0\end{bmatrix},\ G{2}=\begin{bmatrix}0 \;\ 0 \;\ 1\\0 \;\ 0 \;\ 0\\0 \;\ 0 \;\ 0\end{bmatrix},\ G_{3}=\begin{bmatrix}0 \;\ 0 \;\ 0\\0 \ \; 0 \;\ 1\\0 \ \; 0 \;\ 0\end{bmatrix}\]
指数映射
\[\exp(\mathrm{alg}(v(\theta,x,y)))=\exp(\begin{bmatrix}0 \; -\theta \; x\\\theta\ \ \ \; 0 \ \; y\\0 \ \ \ \; 0 \; \ 0\end{bmatrix})\]

定义 1.1

一个拓扑群是一个群 \(G\)，在这个群所在集合上有一个拓扑结构，使得映射\[ f :G \times G \rightarrow G,\ (x,y) \mapsto x^{-1}y \] 连续.(逆和乘法是群中的运算)并且，我们要求这个拓扑空间满足 \(T_1\)（单点集是闭集）或 \(T_2\)(Hausdorff)（任意两个不同的点可以用不交的开集分开）分离性公理.(实际上拓扑群满足\(T_1\) 就会满足 \(T_3\))

我们可以给出另外一个拓扑群的定义：要求群的乘法和逆两个运算连续。容易验证这两个定义是等价的。

拓扑群的结构是很常见的，下面是几个基本的例子：
1.实数配上加法 \((\mathbb{R} , +)\)
2.大于零的实数配上乘法 \((\mathbb{R}^+,\ast)\)
3.挖掉原点的平面配上复数乘法 \((\mathbb{C} - \{ 0 \},\ast)\)
4.将\(n\ast n\)的非退化(行列式不为零的)实（复）矩阵全体看做是空间 \(\mathbb{R}^{n^2} (\mathbb{C}^{n^2})\)的子空间（仅仅是将矩阵的第i行第j列的元素看做\(\mathbb{R}^{i\ast j}\)上的分量），那么矩阵的乘法和求逆是连续的（因为乘法是多项式函数，求逆是一个有理分式函数）。这个空间通常记为 \(GL(n,\mathbb{R}),GL(n,\mathbb{C})\)，而矩阵的行列式函数 \(det(M):GL(n,\mathbb{R}) \rightarrow \mathbb{R}\) 是连续的（因为求行列式是一个多项式函数）
5.所有行列式为1的 \(n\ast n\) 实（复）矩阵，记为 \(SL(n,\mathbb{R}),SL(n,\mathbb{C})\) (Special linear group)

下面的性质是显然的，但是非常重要：

性质 1.2

令\(G​\) 是一个拓扑群（下文中出现\(G​\) ，如果不加以说明，表示一个拓扑群），\(\forall g \in G​\) ，映射 \(L _{g}:G \rightarrow G,x \mapsto gx,R _{g}:G \rightarrow G,x \mapsto xg​\) 是连续同胚，其逆分别为 \(L _{g^{-1}},R _{g^{-1}}​\)$ ​$

这两个映射就是由群中某个元素定义的左（右）作用。不严格地说，这个性质告诉了我们一个拓扑群有很多的自同胚，或者很科普地说，一个拓扑群几何上十分“对称”，我经常把作用想象成一个很对称的东西在转啊转，所以有时候会做噩梦- -而且这个性质还告诉我们一件很重要的事：一个拓扑群的拓扑结构是被其“局部”所决定的，因为任意两点之间存在同胚映射。所以当我们需要考察拓扑群的拓扑性质时，我们常常##### 只需要##### 通过考察包含群幺元\(e\) 的开集来获得信息。

既然这是一个拓扑空间和群结构的混合，那么我们对它就有两方面的关怀：来自拓扑的和群的，下面的几个性质描述了在这两方面它的情况。

性质 1.3

\(G\) 的一个子群配上子空间拓扑还是一个拓扑群。如果\(H\) 是\(G\) 的一个正规子群，并且是拓扑空间中的闭集，那么商群\(G/H\) 配以商空间拓扑是一个拓扑群。\(\Box\)

唯一要说明的是一个\(T_1\) 的空间商掉一个闭子集，商空间是\(T_1\) 的，这在一般情况下不成立，所以我们要求要商掉闭的正规子群。

有一个记号后面总是会出现，先提一下：设\(A,B\) 是\(G\) 中的集合，定义\(AB:= \{ab | a \in A,b \in B \},A^{-1}:=\{a^{-1}|a \in A \}\)

性质 1.4

\(G​\) 商掉一个正规闭子群\(H​\) 得到的商群\(G/H​\) 由性质1.3是一个拓扑群，商映射\(\pi :G \rightarrow G/H​\) 是开映射。

证明：设\(U\) 是\(G\) 中的开集，\(\pi (U)\) 是开集当且仅当\(\pi^{-1} (\pi (U))\) 是开集，而\(\pi^{-1} (\pi (U)) = UH\) 注意到\[UH = \bigcup_{u \in U} uH \] 是开集，故\(\pi (U)\) 是开集。\(\Box\)

性质 1.5

拓扑群子群的闭包是子群，正规子群的闭包是正规子群。

证明：唯一需要用到的拓扑性质是，闭包的连续象包含在连续相的闭包中。\(\Box\)

定义 1.6

一个包含幺元\(e\) 的邻域\(V\) 称为对称的，或者称为对称邻域，如果\(V = V^{-1}\)。

对称邻域是很常见的邻域，任何一个包含幺元的邻域都包含对称邻域并且我们有如下性质，这使得我们在考虑问题时常常只需要关注对称邻域就可以了。

性质 1.7

对任意包含幺元\(e\) 的邻域\(U\) ，存在包含幺元\(e\) 的对称邻域\(V\) ，使得\(VV \subset U\)

证明：设\(G\) 中乘法映射为\(\Phi :G \times G \rightarrow G\) ，令\(\widetilde{V} = \pi_1 (\Phi^{-1} (U)) \cap \pi_2 (\Phi^{-1} (U)), V = \widetilde{V} \cap \widetilde{V}^{-1}\) ，验证\(V\) 开，对称，包含幺元。\(\Box\)

定理 1.8

拓扑群满足\(T_2\) 分离性公理。

证明：一个拓扑空间满足\(T_2\) 分离性公理当且仅当对角集\(\{(a,a) \in G \times G|a \in G \}\) 是闭集，而对角集是在连续映射\(f:G \times G \rightarrow G, (x,y) \mapsto x^{-1}y\) 下的闭集\(\{ e \}\) （由定义\(T_1\) 分离性知这是闭集）的原象，故对角集闭，拓扑群满足\(T_2\) 。\(\Box\)

定理 1.9

拓扑群满足\(T_3\) 分离性公理。

证明：令\(B\) 是不包含幺元的闭集，则存在对称邻域\(V\) 使得\(VV \cap B = \emptyset\) ，那么\(V \cap BV^{-1} = \emptyset\) ，由对称性\(V \cap BV = \emptyset\) 而且\(B \subset BV\) 因为\(e \in V\) \(\Box\)

我们继续拓扑性质的考察,接下来关注的是连通性。

性质 1.10

设\(H\) 是\(G\) 的一个开子群，\(H\) 是开的，则\(H\) 是闭的。

证明：\(H^c\) 是一些\(H\) 陪集的并，而陪集是开的。\(\Box\)

性质 1.11

设\(H\) 是\(G\)中包含幺元的连通分支，那么\(H\) 是正规子群。

证明：因为求逆映射是同胚，故连通分支整个打到连通分支，又\(e^{-1} = e\) ，故\(H^{-1} = H\) ，故\(H\) 对求逆封闭。对乘法封闭因为元素的左作用是个同胚。正规性是因为左作用和右作用都是同胚。\(\Box\)

这些证明看起来都反映了拓扑群中非常强的对称性，而上述性质告诉我们拓扑群的连通分支之间是互相同胚的。

我们来讲一点稍微有趣的东西作为这篇乱七八糟的介绍的结尾吧，关于路径连通性和基本群。

考虑\((G,e)\)中（取定幺元为基点的拓扑空间）的回路\(f,g:(S^1,(1,0)) \rightarrow (G,e)\)，我们在\(f,g\)之间有两种乘法：拓扑空间中路径的连接，记作\(f \cdot g\)，还有在群中的逐点相乘，记作\(fg\)。第一种乘法是我们比较熟悉的乘法，在同伦等价类中就是基本群的乘法。接下来说明实际上在同伦意义下，两种乘法是等价的。

性质 1.12

设\(f,g\)是\((G,e)\)中的回路，则\(fg\)也是连续回路。

证明：因为\(f \times g:(S^1,(1,0)) \rightarrow (G\times G,(e,e))\)是连续的。\(\Box\)

引理 1.13

设\(f_1,f_2,g_1,g_2\)是\((G,e)\)中的回路，如果\(f_1 \backsimeq f_2\)，则\(f_1g \backsimeq f_2g\)，同样，如果\(g_1 \backsimeq g_2\)则\(fg_1 \backsimeq fg_2\)

证明：在性质1.11的证明中，\(f_1 \times g \backsimeq f_2 \times g\)，由此构造出\(f_1g \backsimeq f_2g\)。另一半同理。\(\Box\)

引理 1.14

设\(f,g\) 为\((G,e)\) 中的回路，\(f \cdot g \backsimeq fg\)

证明：\(f\backsimeq f\cdot e\backsimeq e\cdot f , g\backsimeq e\cdot g\backsimeq g\cdot e\)，由引理1.12，对这三个回路逐点相乘，我们得到\(fg \backsimeq f\cdot g \backsimeq g\cdot f\) \(\Box\)

基础知识

实际访问网站时，确定从哪台服务器获取网页，对于计算机来说是通过IP地址实现的。由于IP地址是数字形式，为了便于人们记忆，所以使用了域名和网址机制。

域名->IP地址：在访问网络时，网址会被发送到DNS服务器，然后由DNS服务器返回我们所要访问的服务器IP地址，从而进行访问。这是DNS的工作原理，也是之所以需要设置域名解析的原因。我们要在DNS服务器上设置域名解析，让域名指向服务器空间所在的IP地址。

IP地址->域名：一台服务器上可能有多个网站，或者是子网站，他们共用一个IP地址。这时，我们请求的数据被服务器接收到，服务器需要根据判断网址中的域名来判断返回哪个站点的数据。所以我们还要在自己的服务器空间上绑定自己的域名。即域名和服务器空间的绑定关系是双向的。

配置方法

域名解析的配置分为三个部分内容：

  • 给域名设置DNS服务器（在域名提供商网站中进行）
  • 在DNS设置域名解析（在DNS服务提供商网站中进行）
  • 在服务器空间设置绑定域名（在服务器空间面板中进行）

这三者没有先后关系，但是都设置好才能完成域名解析。

给域名设置DNS服务器

  • 主机记录、记录值。设置由 主机记录 指向 记录值。
  • TTL。TTL是DNS服务器上刷新记录的时间间隔，如果需要频繁更改IP，记录时间可以短一些，否则应该设置长一点，这样一定程度上会加快访问速度。不了解的话，直接用默认值即可。

4.1 Features of WordPress

WordPress是一个被广泛使用的网站框架，主要用于博客类网站，但也被一些站点做成资讯类网站、购物网站等。

我们可以感受到，使用了WordPress这一框架，很轻松的就搭建了一个功能完整的个人博客网站，完全不涉及任何代码。这一篇我们一起熟悉WordPress的基本应用。你会发现，WordPress用起来就和装扮QQ空间一样简答方便。

由于我们的WordPress是安装在我们租用的远程服务器上的，在对网站进行管理的时候，一切操作都通过网络发送到服务器端处理，处理完在通过网络发送到我们的电脑上，整个过程都是在浏览器中进行，和QQ空间装扮一样。通过网络传输自然有延迟，尤其是国外的服务器，并且我们的服务器流量也有限。

更好的办法是，让WordPress运行在自己的电脑上，并进行配置，配置好了，再转移到服务器上。

4.2 Installing local WordPress

让本地计算机支持WordPress，需要安装PHP、MySQL等环境，但是这里已经有软件为我们全部做好了，例如USB WebServer。

把WordPress文件解压到root子目录中，浏览器访问 http://localhost 就可以打开本地的WordPress页面了。当然还需要进行数据库的配置。

如果你只是想学习WordPress相关知识而不打算实际建网站，也可以用这种方式来学习。

4.10 Migration

在本地设置好主题等配置后，需要上传到服务器端。WordPress的数据分两个地方保存，一个是WordPress文件夹，另一个是数据库。由于不同的插件数据保存的位置格式等可能不一样，如果直接把文件覆盖到服务器，并把本地数据库原封不动导进在线服务器的数据库，可能会出现一些问题。

后面还会讲到服务器的更换，域名的更换等，数据迁移都是必要的。网上有很多参考资料，但是实际使用时我还是遇到了一些问题，解决起来也有点麻烦，对于初学者不好理解。同时考虑到本篇主要讲基本配置，数据迁移就先不做介绍了。

所以本地数据迁移到服务器，最保险的方法是：把安装的主题和所有插件打包起来上传到服务器（也就是themes和plugins文件夹中的文件，这两个文件夹中不会用来保存配置文件）；然后在服务器端，对照本地的设置再配置一遍。

值得注意的是，如果你在本地WordPress中编辑页面和文章，数据会被保存在数据库中，需要通过数据库才能迁移。如果文中还有图片，图片的引用地址是 http://localhost ，不能直接迁移到服务器，还需要替换数据库中数据的网址字段，本篇暂不介绍。所以建议直接在服务器端进行编辑。

使用Wordpress的不安

WordPress 的一些功能可能比较合大多数用户，但对于我来说，还是造成了不安。比如：

  1. 上传图片占用ID：上传图片会占用 post id，而碰巧我又是使用 post id 作为固定链接；
  2. 自动保存草稿和修订记录占用ID：wordpress一个很“贴心”的的功能就是自动保存文章草稿和修订记录。这些功能无法在设置中关闭，且于代码深度整合，对于我们这种不懂代码的很不易修改，而这些记录会占用很多ID。最主要是wordpress官方似乎会将自动保存和修订记录功能更深入化的迹象。
  3. 更换固定链接的代价：如果将 post id 更换为 post name 别名，则会影响搜索引擎搜录，且内部的链接好多都要重新设置。更换固定链接的代价与单纯的ID洁癖相比是否值得？！

对于自动保存文章草稿和生成多个版本的修订记录我个人觉得是个很鸡肋的功能。先说自动保存，你要自动保存就保存到当前ID的 post 中吧，非要另开一个ID来保存；修订记录也是，你开一贴编辑，四分之一柱香不到的时间就保存了若干修订历史记录，这得占用多少资源啊，也不利于节能环保什么的。好吧，你说有人需要，那么你就加个设置选项开关啊，真是不懂！

图片上传到WordPress好处

WordPress中上传的图片不仅仅是附件，而是一种文章类型。如果使用 WordPress 上传一张图片, 数据库中会生成一条类型为 attachment 的 post 数据。因些图片的编辑修改删除都很方便，便于管理，网站能访问图片就能访问，不会受到图库的影响。

很多网友分析 wordpress 内建图库, 如果图片相关性和文字描述做得足够好, 图片很容易被搜索到, 而且图片搜索的转化率很高，所以使用 WordPress 内建图库作为文章的大图页面, 也是 WordPress SEO 中重要的一环。但使用 wordpress 的内建图库时须注意：

  • 填写清晰, 带关键字的图片 alt 和 title；
  • 将图片放在文章中相关性高的段落前后；
  • 如果使用多语言, 在不同的语言版本使用不一样的图片描述 (否则搜索引擎无法定位主页面)；
  • 避免使用超大的图片, 以影响页面加载, 图片备份和网站流量。

1. Host machine

1.1 Virtual hosting or virtual private server (VPS)

Virtual hosting is a method for hosting multiple domain names (with separate handling of each name) on a single server (or pool of servers). This allows one server to share its resources, such as memory and processor cycles, without requiring all services provided to use the same host name.

A Virtual Private Server (VPS) is a virtual machine sold as a service by an Internet hosting service. A VPS runs its own copy of an operating system (OS), and customers may have superuser-level access to that operating system instance, so they can install almost any software that runs on that OS. For many purposes they are functionally equivalent to a dedicated physical server, and being software-defined, are able to be much more easily created and configured.

As we can see from above virtual hosting can be convenient but with limited functions and less freedom. As for a VPS, we need to setup particular servers and configure them all by ourselves but we can do nearly everything we want as long as it is available for a Linux machine. Certainly, I chose VPS for my purpose of learning.

1.2 Choosing and purchasing a VPS

First, we need to considering several aspects listed below:

  • Whether dedicated IP addresses are provided
  • Convenience of changing IP (really important because our IP may be banned by GFW)
  • Quality of IP (may be GFW or some of the network servers in mainland China had already banned quite a lot of IPs of the same range)
  • Pricing, of course most importantly
  • Hardware configuration
  • Location & whether it can be changed
  • Technology (KVM better than OpenVZ)
  • Panel provided
  • Method of payment
  • ……………..

Analyzing these aspects, we recommend following businesses:

  • Bandwagon: cheap, speedy, high quality, pay $2.8 if you need to change IP, a lot of IP from there are banned
  • Vultr: cheap, free to change IP, but few of them are not banned currently, hard to configure
  • DigitalOcean: price a little higher
  • Cyanode: not known by most people
  • Linode: price higher

For me, I chose to purchase a $4.99 per month (1G, 20G, several locations) and another $19.99 per year (512M, 10G, Los Angeles only, actually changing IP twice up to now) on Bandwagon and prepaid $25 for my Vultr accounts (a huge mistake).
